## installation
1. Clone the repo: `git clone https://github.com/Originaalne1/Organizer`
2. Run the installer: `python install.py`
3. Restart your terminal.
4. Type `utils help` to get started!

## new commands
- `utils log` — show the latest organizer activity
- `utils log <count>` — show the last N log entries
- `utils find <keyword> [path]` — search for files by name
- `utils status [path]` — report file category counts in a folder
- `utils organize --report` — show a summary after organizing
